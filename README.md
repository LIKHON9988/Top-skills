# Alpha!vite (Firebase)

Project name: Assign-Ment-09

Purpose:
Alpha!vite is a modern, mobile-responsive skill-sharing platform where learners and mentors connect to exchange skills. It features Firebase authentication (Email/Password and Google), user profile management, searchable skill listings, and a clean UX built with React + Vite + Tailwind CSS.

Live URL: https://top-skills-09.netlify.app/

Key Features:

1. Mobile-responsive design across all pages (Home, Login, Signup, Profile, SkillDetails)
2. Firebase Authentication: Email/Password signup and login, Google sign-in
3. Password reset (ResetPassword page)
4. Robust Google sign-in with popup and redirect fallbacks
5. Profile page: view and edit display name and photo URL
6. Search input in the Home hero to filter skills
7. Skill listing and details from local JSON data
8. Testimonials and CTA sections on the Home page
9. Dark mode toggle in the Navbar
10. Friendly error handling and user feedback via toasts
